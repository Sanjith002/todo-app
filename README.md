# 📝 Todo App

A full-stack Todo Application built with Spring Boot, PostgreSQL, JWT Authentication, and a responsive HTML/CSS/JavaScript frontend.

## 🌐 Live Demo

https://todo-app-49f.pages.dev/

## 🚀 Features

* User Registration
* User Login with JWT Authentication
* Create Todos
* View Todos
* Mark Todos as Completed
* Delete Todos
* Persistent Storage with PostgreSQL
* RESTful API Backend
* Responsive UI

## 🛠️ Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript (Vanilla JS)

### Backend

* Java 17
* Spring Boot
* Spring Security
* Spring Data JPA
* JWT Authentication

### Database

* PostgreSQL

### Deployment

* Frontend: Cloudflare Pages
* Backend: Render
* Database: Render PostgreSQL
